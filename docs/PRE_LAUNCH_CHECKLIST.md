# NRAPA Members Portal — Pre-Launch Checklists

Pre-launch work is split into two task lists:

| Checklist | Who | Purpose |
|-----------|-----|---------|
| **[PRE_LAUNCH_CHECKLIST_NRAPA_STAFF.md](PRE_LAUNCH_CHECKLIST_NRAPA_STAFF.md)** | NRAPA staff | Business, domain access, service provider change, member import verification, content & document sign-off (wording, layout), Learning Centre & tests, communications, support |
| **[PRE_LAUNCH_CHECKLIST_DEV.md](PRE_LAUNCH_CHECKLIST_DEV.md)** | Developer / agency | Technical deployment, infrastructure, import run, payments, security, testing member flows, monitoring |

Use the **NRAPA staff** list when handing tasks to the customer. Use the **Dev** list for your own pre-launch work.
